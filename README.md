# 🧹 Automated Data Wrangling Pipeline

This repository contains a fully automated Python script designed to ingest highly unstructured, corrupted client data and output a pristine, analysis-ready dataset. 

As a Data Science student and freelance Data Analyst, I built this pipeline to eliminate the hours of manual data entry required to fix broken business spreadsheets.

### ⚠️ The Problem
Businesses frequently generate or export data containing:
* **Corrupted Numeric Columns:** Text mixed with numbers (e.g., `$15.99`, `FREE`).
* **Broken Date Formats:** Inconsistent structures (e.g., `15/01/2023` vs `Jan 16th 23`) that break analysis tools.
* **Missing & Duplicate Data:** Blank ID fields and duplicate transaction logs.
* **Unstructured Text:** Trailing spaces, inconsistent capitalization, and missing names.

### ✅ The Solution
A dynamic, automated Pandas script that instantly cleans the data without human intervention. The script intelligently:
* Drops critical nulls and standardizes duplicate entries.
* Employs forward-filling (`.ffill()`) to logically patch missing dates.
* Coerces corrupted text out of numeric columns.
* Formats strings to title case and locks dates into standard `YYYY-MM-DD` text format to prevent `NaT` errors.

### 📂 Repository Contents
* `messy_client_data.csv`: A sample dataset simulating severe real-world data entry errors.
* `clean_client_data.csv`: The flawless output generated by the pipeline.
* `data_cleaner.ipynb`: The core Python script utilizing Pandas to perform the automated cleaning.
* `creating_messy_data.ipynb`: Creating the sample messy dataset.
